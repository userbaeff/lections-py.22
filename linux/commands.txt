ctrl+alt+t - команда открытия терминала
MacOS (B спотлайт вводим Terminal)
win+space
 ~ - символ домашней директории (папки)
 / - символ корневой директории (папки)
 .. -  обазначает предыдущую директорию
 . -  обазначает текущую директорию
 
 
 ls (list)-список всех дериктории и файлов в текущей директории
 ls -a - показывает список всех файлов и директории включая скрытых
 ls -l - показывает список файлов и директории
 ls -la - показывает список всех файлов и директории + инфо о них 
 cd (change directory) - переход с текущей директории на указанную cd Desktop
 mkdir (make directory) - создает директорию mkdir Linux
 mkdir .(name of file) - создает скрытые файлы
 touch - создает указанный файл touch commands.txt (extension-расширение (.txt, .docs, .py))
 pwd (print working directory) - выводит путь до текущей директории
 rmdir (remove direсtory) - удалять директорию (не удаляет не пустую директорию)
 ctrl+shift+c - копирует в терминале
 ctrl+shift+v - вставляет в терминале
 clear, ctrl+l - очищает в терминале
 rm (remove) - удаляет файлы (rm название файла)
 rm -rf - принудительное удаление файлов или директории (использовать крайне редко)
 mv (move) - действие с файлом (перемещение: название файла name/, переименование: название файла название файла)
 cat - выводит содержимое файла
 nano - открывает графический интерфейс в терминале для редактирования файла
 *если нет такого файла, то создасться файл с таким же именем
 echo - команда для быстрой записи в файл через терминал echo 'текст то,что хотим написать' > name of file
 cp (copy) - команда копирования файла на нужную директорию cp <название файла> <название директории куда надо скопировать>
